
# MMG Product Engine v1.5.0

Status: Canonical Draft

## Purpose
Defines the production standard for every MMG Shopify product and digital asset.

## Product Families
- AI Mastery Series
- Creator's Bible
- Micro Packs
- Reset Series
- Merchandise

## Canonical Product Record
Each product maintains:
- Product title
- SKU/ID
- Shopify handle
- Price
- Product description
- SEO title
- Meta description
- Cover image
- Gallery images
- Digital download (if applicable)
- Release notes
- QA checklist
- Version history

## URL Standard
Use short handles:
- /products/creators-bible
- /products/ai-prompting-for-beginners

## Release Package
Every product release includes:
- Product source
- Images
- Metadata
- QA
- Release notes

## QA
- Pricing verified
- Links verified
- Images optimized
- Mobile layout verified
- Download tested
- SEO fields completed

## Golden Master
Approved products become the canonical production version until superseded by a new approved release.
